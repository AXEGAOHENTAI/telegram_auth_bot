# Telegram Steam Guard Bot

Телеграм бот для получения Steam Guard кодов по команде с системой авторизации пользователей.

## 📋 Описание

Этот бот позволяет получать Steam Guard коды через Telegram команды. Он интегрируется с существующей функцией `getOneTimeCode` и предоставляет удобный интерфейс для управления Steam Guard кодами с возможностью ограничения доступа только для авторизованных пользователей.

## 🔐 Система авторизации

Бот поддерживает два режима работы:

### 🔒 Режим авторизации (по умолчанию)
- Доступ только для авторизованных пользователей
- ID пользователей хранятся в файле `.env`
- Неавторизованные пользователи получают сообщение об отказе в доступе

### 🔓 Режим свободного доступа
- Доступ для всех пользователей
- Можно включить для тестирования

### Управление пользователями

```bash
# Запуск скрипта управления
python manage_users.py
```

**Доступные действия:**
1. Показать список пользователей
2. Добавить пользователя
3. Удалить пользователя
4. Переключить режим авторизации

### Получение ID пользователя

1. Отправьте `/myid` боту в Telegram
2. Или используйте @userinfobot в Telegram
3. Скопируйте полученный ID

## 🤖 Использование бота

После запуска используйте команды в Telegram:

- `/start` - Начать работу с ботом
- `/code <имя_аккаунта>` - Получить Steam Guard код
- `/help` - Показать справку
- `/myid` - Показать ваш ID (для настройки авторизации)

### Примеры команд:
```
/code my_steam_account
/code test_account
```

## 🐛 Устранение неполадок

1. **Ошибка импорта модулей**: Убедитесь, что установили все зависимости
2. **Бот не отвечает**: Проверьте правильность токена в файле `.env`
3. **Код не генерируется**: Убедитесь, что файл `Steamguard.txt` существует
4. **Ошибка доступа к файлам**: Проверьте права доступа к папкам `accounts/` или `accountStash/`
5. **Проблемы с настройкой**: Используйте `python setup_bot.py` для автоматической настройки
6. **Пользователь не может получить доступ**: Проверьте ID пользователя в настройках авторизации
